\subsubsection*{template$<$typename T$>$\newline
class Lista\+Simple$<$ T $>$}
Implementa una lista enlazada simple de elementos de tipo genérico T. 

Esta clase plantilla ofrece funcionalidades básicas de una lista enlazada, como agregar, eliminar y buscar elementos, además de obtener información sobre la lista.
